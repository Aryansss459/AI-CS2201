# Artificial Intelligence Course Assignments

This repository contains assignments completed as part of the Artificial Intelligence course.
Each folder represents a separate assignment with the code and files used to solve the given problems.

## Repository Structure

```
Assignment-1/
Assignment-2/
Assignment-3/
AQI_Project/   (included as a submodule reference)
```

## Topics Covered

Some of the topics explored in these assignments include:

* Intelligent agents and environments
* Search algorithms
* Problem solving in AI
* Basic AI implementations using Python

## Project Reference

The **AQI_Project** folder links to an external project related to Air Quality Index calculation.
It is included here as a submodule reference to a project I contributed to.(My team)

## How to Run

Most assignments can be run using Python:

```
python filename.py
```
